# Directives

This folder contains all SOPs (Standard Operating Procedures) for the EWUMart project.

Each directive is a Markdown file that defines:
- **Goal**: What the task accomplishes
- **Inputs**: What data or parameters are required
- **Tools/Scripts**: Which `execution/` scripts to use
- **Outputs**: Expected deliverables
- **Edge Cases**: Known gotchas, limits, and recovery steps
